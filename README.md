# 万能電卓アプリケーション

高機能でシンプルな電卓アプリケーションです。数式の計算や多彩な機能をサポートし、使いやすさを追求したデザインを採用しています。

## 特徴
- 四則演算、平方根、指数計算などの基本的な計算機能
- メモリ機能による計算結果の保存
- ユーザーインターフェイスのデザインにこだわり
- クロスプラットフォーム対応（Windowsでの使用を推奨）

## 使用方法
- アプリケーションを起動すると、簡単なインターフェイスが表示されます。
- 数字ボタンと計算ボタンをクリックして、計算を行います。
- 結果は画面に表示され、メモリ機能を使って保存・読み込みが可能です。
